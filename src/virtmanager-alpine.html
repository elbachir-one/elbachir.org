Virt-manager installation on Alpine Linux

Installation

doas apk add virt-manager libvirt qemu qemu-system-x86_64 dnsmasq bridge-utils qemu-openrc libvirt-daemon dbus polkit nano

User Setup

doas adduser <your_username> kvm
doas adduser <your_username> libvirt

Configuration

Set default URI

Edit /etc/libvirt/libvirt.conf:

doas nano /etc/libvirt/libvirt.conf

Uncomment and set:

uri_default = "qemu:///system"

QEMU Permissions

Edit /etc/libvirt/qemu.conf:

doas nano /etc/libvirt/qemu.conf

Uncomment and set:

user = "<your_username>"
group = "libvirt"

Enable Services

doas rc-update add libvirtd default
doas rc-update add dbus default
doas rc-update add polkit default

Reboot the System:

doas reboot
